Utilizei o Xamp para fazer requisição do Less.

Utilizei LESS no CSS, utilizei alguns efeito gradient e shadow para dar uma estilizada na tabela e no titulo.

Dentro do Javascript : 

Criei um button para chamar uma função: inserir(), que dentro dessa função faz a chamada para as funções :

recebeValores() - recuperar os dados do exercicios (hora, modalidade, data)

formataData() - formarta a data para o padrão brasileiro DD/MM/YYYY

inserirLinha() - Inserir a linha na tabela com os dados é um button para deleta a linha

somaTotal() - Recebe o paramentro de horas de exercicio e converte para inteiro depois e adicionado a um contador para retorna o total de hora da tabela.

MostrarTotal() - Atualizar o valor total de exercicios tanto os que foi adicionado ou removido da tabela.

Na quarta coluna da tabela tem um botão de remover com duas chamadas de função:

delLinha() - Remove a linha correspodente ao botão

subtrair() - Subtrai o valor de horas da linha do botão e atualizar.
